Pieces of a website

1. HTML

Hypertext Markup Language, a standardized system for tagging text files to achieve font color, graphic and hypertext effects in creation of World Wide Web pages.

  • 5 Common HTML Elements:
  • Head, body, div, li and ul.

2. CSS

Cascading Style Sheets, a computer language for laying out the structure of a webpage. The language contains coding elements which are subsequently called CSS files. Ultimately, the design of the webpage.

  • 5 Common CSS properties:
  • Color, line thickness, font, point, and margin.

3. JavaScript

JavaScript is a scripting or programming language which allows the user to implement complex features on webpages -- every time a webpage does more than just sit there and display static information. Timely content updates, interactive maps, animated 2D/3D graphics and scrolling videos jukeboxes.

  • List 5 variable types, one of each words, whole numbers and single letters
  • String, number, bigint, boolean, and undefined.
